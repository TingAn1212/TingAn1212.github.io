《纪念》

Short Film 

    亲人的爱，如夜空中耀眼的星，指引我们回归初心，觉醒曾经遗忘的梦想。

    This is a film submission for National Chinese Short Film Competition, telling a story of a girl, her grandmother, and dream.

Made In Heaven

Unity Project 

    Made In Heaven is a unity game built around the concept of limited reaction time, as the game continues, the game would speed up drastically, making the difficulty increase.

RdeV

Website 

    rdev.x10.mx is the official website for RdeV, a school club focused on software development. I also participated in the creation of RdeV. This website serves as an introduction to RdeV, it also has backend capabilities to host interactive notes.

RVHS Game Jam

Website/Event 

    RVHS Game Jam is a school-wide game-making competition which encourages students to develop games. I participated in the hosting of this event as well as created the website.

Dodogama

Python App 

    Dodogama is a desktop-pet app coded with python tkinter, where users can interact with the pet by clicking. The pet would also shuffle through a random list of actions to perform when idle. (This app is still being updated)

Tello Control

Android App 

    This is an app coded in kivy that works as a remote control for Tello drones. This controller offers a lot of unique and creative ways to control a drone, such as voice control and motion control using the accelerometer.

Remofe

Website 

    Remofe is a tool to convert all f-string into string.format() in a python file.

    This is to help integrate python code implemented in newer version into a older system such as a graphing calculator without the need to edit the code manually.
